About

playnode가 3년 만에 다시 여러분을 찾아 갑니다.

playnode는 국내 node.js community인 octobersky.js 의 주최로 2012년 11월 20일에 처음으로 열린 node.js conference로서 국내외 연사들의 경험과 노하우를 듣고 다양한 이야기를 나눌 수 있는 자리입니다. 또한, 여러 분야의 사람들이 한자리에 모일 수 있는 즐거운 개발자 축제 같은 행사이기도 합니다.

Node.js는 올해 Node.js 재단을 설립하고 io.js와 다시 합친 뒤 엔터프라이즈 Node.js를 지원하기 위해서 LTS 버전을 릴리즈 하는 등 새로운 단계에 진입했습니다. 그동안 개발이 느리고 io.js가 분리되는 등의 이슈가 있었지만, 지금은 Node.js 재단 아래 커뮤니티 주도로 모여서 이전보다 더욱 활발하게 개발이 진행되고 있고 여기에 IBM, 레드햇, Microsoft, Intel 등에서도 적극적으로 지원하고 있습니다. 이런 상황에서 국내에서 Node.js에 대한 경험을 공유하고 앞으로 Node.js가 나아가는 방향에 대해서 이야기 해 볼 수 있는 좋은 자리입니다.

2015년을 기점으로 octobersky.js 로부터 분리되어 독자적인 브랜드로서 새롭게 출발합니다.

    npm@>=3

    Kat Marchán@npm inc

    현재 Node.js 4.2.x LTS 버전에 포함된 npm 2.x의 다음 버전인 npm v3에 대해서 npm cli 개발자가 직접 설명합니다. npm@3에서 프론트앤드 라이브러리 지원을 위한 준비작업인 peerDependencies의 변경과 flat하게 설치하는 의존성 등 npm@3가 어떻게 달라졌는지를 설명합니다.
